Pseudo elementos

Estos elementos se usan para aplicar estilos a una parte de un elemento

First_line

El primero es ::first-line, el cual aplica estilos a la primera línea de un párrafo, si las dimensiones de la página cambian y en la primera línea de ese párrafo se agrega más texto de las líneas posteriores, esas palabras también cambiarán según las especificaciones dadas en first line, si por el contrario, se reduce, solo a las letras que estén en la primera línea se les aplicará el estilo asignado, para ver mejor este estilo y tener más líneas de texto, se aumenta el tamaño de este párrafo

First_letter

El segundo elemento es ::first-letter, el cual selecciona la primera letra de ese párrafo

Before

También está el elemento before, el cual coloca, depende lo que se coloque en el código de css, un objeto antes del elemento principal

After

En contraste con before, está el elemento after, el cual coloca, depende lo que se coloque en el código de css, un objeto después del elemento principal

Selection

Y también está el elemento selection, el cual aplica el estilo al elemento cuando se selecciona con el mouse el texto

Box Model

El box model de CSS es el que indica el como se posicionan los elementos de la pagina web, todo elemento va a tener un margin, un border, un padding y finalmente el contenido

Margin

Esta propiedad genera un margen o espacio por fuera de los elementos, se puede definir individualmente en donde se quiere aplicar el margen

  • Margin-top
  • Margin-bottom
  • Margin-left
  • Margin-right

Pero estas características se pueden indicar en una sola linea con solo margin recibiendo 4 valores, el primero Margin-top, el segundo Margin-right, el tercero Margin-bottom y el cuarto Margin-left

Esté texto tiene la propiedad de: margin: 15px 20px 25px 30px;

Otra característica de margin es que si en vez de 4 valores, se coloca 1 valor, entonces todos los lados tendrán ese mismo valor

Esté texto tiene la propiedad de: margin: 10px;

Y si margin recibe 2 valores, entonces el primer valor será el margin vertical, y el segundo será en margin horizontal

Esté texto tiene la propiedad de: margin: 15px 50px;;

Padding

Padding genera un margen o espacio por dentro de los elementos y se pueden añadir las mismas características que con margin

  • Padding-top
  • Padding-bottom
  • Padding-left
  • Padding-right

Igualmente se pueden indicar en una sola línea con solo padding recibiendo 4 valores, el primero Padding-top, el segundo Padding-right, el tercero Padding-bottom y el cuarto Padding-left

Esté texto tiene la propiedad de: padding: 30px 25px 20px 15px;

E igual que margin, si padding recibe solo un valor entonces todos los lados tendrán ese mismo valor

Esté texto tiene la propiedad de: padding: 10px;

Y si recibe dos valores entonces el primer valor será el padding vertical, y el segundo será en padding horizontal

Esté texto tiene la propiedad de: padding: 15px 50px;

Padding y Margin

Mezclando estas dos propiedades con border, se observa que está ultima se ubica entre margin y padding

Esté texto tiene ambas propiedades, un margin de 30px y un padding de 30px combinadas junto con border
